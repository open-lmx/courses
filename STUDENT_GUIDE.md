# Student Guide

## Getting Started

### 1. Create Account
Sign up with your email or use SSO:
- Google
- GitHub
- Generic OIDC (work account)

### 2. Set Your Goal
When you first log in, set your learning goal:
- "Get a job in AI"
- "Learn new skills"
- "Advance my career"

### 3. Browse Courses
```
/courses - Browse all courses
/skills - View skill library
/learning-paths - Recommended paths
```

## How to Learn

### Enroll in a Course
1. Find a course
2. Click "Enroll"
3. Start learning!

### Track Progress
Your dashboard shows:
- Courses enrolled
- Skills learned
- Job preparation status

```bash
GET /api/recommendations?goal=get job
```

This returns courses matching your goal.

## Job Preparation

### Link Skills to Jobs
After completing courses, check job readiness:
```
/jobs - View job openings
```

Jobs show required skills. Your completed courses = matching skills = qualified!

### Example
```
Course: Building AI Agents
Skills: PROG (Advanced), DEVO (Intermediate)
↓
Job: AI Engineer (requires PROG, DEVO)
✅ Qualified!
```
