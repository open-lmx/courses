# LangGraph + Next.js Course

Build code-first autonomous AI systems using LangGraph Deep Agent Framework and Next.js.

## About

OpenAutonomyx (OPCI) Private Limited - A private limited company incorporated under the Companies Act, 2013.

**Faculty:** Chinmay Panda

## Course Curriculum

- **Module 1:** Foundations of AI Systems
- **Module 2:** Core Agent Development
- **Module 3:** Next.js Integration
- **Module 4:** Advanced Patterns
- **Module 5:** Production Deployment

## Features

- NextAuth.js authentication
- Docker deployment (OCI-compliant)
- CI/CD with GitHub Actions & GHCR
- Lago billing integration
- Responsive design

## Quick Start

```bash
npm install
npm run dev
```

## Docker
